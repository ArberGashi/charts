Vector API required for spatial renderers

The 3D and spatial showcase requires the JDK Vector API. Start the verifier with --add-modules jdk.incubator.vector to enable spatial charts.

Maven / CLI: JAVA_TOOL_OPTIONS="--add-modules jdk.incubator.vector" mvn -pl arbercharts-visual-verifier spring-boot:run IntelliJ / IDE: Add JVM option: --add-modules jdk.incubator.vector
