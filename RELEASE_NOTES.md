# ArberCharts Core 1.0.0 Release Notes

Release: 1.0.0  
Scope: `arbercharts-core` (final release)

---

## Highlights

- Fluent API (`ArberChartBuilder`) for fast chart assembly
- Stable public API map (`PUBLIC_API.md`)
- Theme-aware rendering with dark/light defaults
- Legend system with overlay/docked modes
- High-precision crosshair overlay
- Domain grid layers (default/medical/financial/analysis)
- Export services for PNG (built-in) and SVG/PDF (optional deps)
- Visual and performance baselines in `QUALITY_REPORT.md`

---

## Compatibility

- Java 25+
- Swing-based UI components
- Optional FlatLaf integration via UIManager defaults
