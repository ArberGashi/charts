# ArberCharts Binary Bundle

Dieses Verzeichnis ist fuer GitHub-Publikation und enthaelt die offiziellen 2.0.0 Artefakte.

## Dateien

- `arbercharts-core-2.0.0.jar`
- `arbercharts-demo-2.0.0.jar`
- `arbercharts-starter-2.0.0.jar`
- `run-demo.sh` (macOS/Linux)
- `run-demo.bat` (Windows)
- `SHA256SUMS.txt`

## Start

```bash
java --enable-native-access=ALL-UNNAMED --add-modules jdk.incubator.vector -jar arbercharts-demo-2.0.0.jar
```

oder:

```bash
./run-demo.sh
```
