# ArberCharts 1.7.0-LTS Release Notes

## Highlights
- Long-term support release focused on stability, documentation, and bridge readiness.
- Swift bridge now ships as macOS arm64 `.xcframework` (iOS/iPadOS not available in 1.7.0-LTS).
- Starter bundle provides a single JAR for JVM clients (core + swing + server).

## Release Assets
- JVM Starter: `arbercharts-starter-1.7.0-LTS.jar`
- Compose: `arbercharts-compose-bridge-1.7.0-LTS.jar`
- Qt (macOS arm64): `arbercharts-qt-bridge-macos-1.7.0-LTS.zip`
- Swift (macOS arm64): `arbercharts-swift-bridge-macos-1.7.0-LTS.zip` (includes `.xcframework` + core dylib)

## Platform Notes
- Swift bridge is macOS arm64 only in v1.7.0-LTS.
- iOS/iPadOS targets are not supported in this release.

## Support Policy
- Only v1.7.0-LTS is supported. Older versions are end-of-life.
