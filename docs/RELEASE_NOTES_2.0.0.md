# ArberCharts 2.0.0 Release Notes

Release date: 2026-02-14

## Summary

ArberCharts 2.0.0 is the active and only maintained release line.
The product focus is JVM-first with Java 25.

## Platform Scope

- `arbercharts-core`
- `arbercharts-swing-bridge`
- `arbercharts-server-bridge`
- `arbercharts-spring-boot-starter`
- `arbercharts-starter`
- `arbercharts-visual-verifier`
- `arbercharts-demo`

## Technical Highlights

- Zero-GC rendering policy in hot paths
- Lock-free circular model for realtime streams
- Java 25 baseline with Vector API readiness
- Spring Boot 4.0.2 starter integration
- Visual Verifier HTTP APIs and catalog tooling

## Renderer Coverage

ArberCharts 2.0.0 ships with 157 production renderers.
