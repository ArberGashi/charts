# ArberCharts

ArberCharts is a premium Java charting framework focused on performance, clarity, and high‑end presentation.  
Built for production applications with a fluent API and extensive renderer coverage.

## Downloads (Final Release)

Release assets are published on GitHub Releases:
- Core JAR: `arbercharts-core-1.0.0.jar`
- Demo JAR: `arbercharts-demo-1.0.0.jar`
- macOS DMG: `ArberCharts Demo-1.0.0.dmg`

## Quick Start (Demo)

```bash
java --enable-native-access=ALL-UNNAMED -jar arbercharts-demo-1.0.0.jar
```
