Quick Start

Fluent API mit ArberChartBuilder

Beispiel fuer einen minimalen Chart-Panel-Aufbau (JVM).

ArberChartPanel panel = ArberChartBuilder.of()
    .addLayer(model, new LineRenderer())
    .withTheme(ChartThemes.getDarkTheme())
    .buildPanel();
Examples

Produkt-Integration in der Praxis

Standard Chart

ChartModel model = ModelBuilder.of("Series A")
    .setXY(0, 2)
    .setXY(1, 4)
    .setXY(2, 6)
    .setXY(3, 5)
    .setXY(4, 7)
    .build();

ArberChartPanel panel = ArberChartBuilder.of()
    .addLayer(model, new LineRenderer())
    .withTheme(ChartThemes.getLightTheme())
    .buildPanel();

Financial Overlay

ChartModel model = ModelBuilder.of("AAPL")
    .setOHLC(0, 100, 110, 98, 105)
    .setOHLC(1, 105, 112, 104, 110)
    .build();

ArberChartPanel panel = ArberChartBuilder.of()
    .addLayer(model, new CandlestickRenderer())
    .addLayer(model, new MACDRenderer())
    .buildPanel();

Medical Sweep

ChartModel model = ModelBuilder.of("ECG")
    .setXY(0, 0.12)
    .setXY(1, 0.18)
    .setXY(2, -0.05)
    .build();

ArberChartPanel panel = ArberChartBuilder.of()
    .addLayer(model, new SweepEraseEKGRenderer())
    .buildPanel();
