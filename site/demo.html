Quick Start

Fluent API mit ArberChartBuilder

Beispiel fuer einen minimalen Chart-Panel-Aufbau (JVM).

ArberChartPanel panel = ArberChartBuilder.of()
    .addLayer(model, new LineRenderer())
    .withTheme(ChartThemes.getDarkTheme())
    .buildPanel();
