Advanced Analysis & Statistics

Engineering Systems

Integration

Einbindung in eigene Software

Kurze, robuste Integration von Desktop-Swing bis Spring Boot.

Swing Desktop Integration

ArberCharts fügt sich in jede Swing-Oberfläche ein.

  • Panel in JFrame, Tabs oder Docking-Layouts.
  • Themes und Grid-Layer frei kombinierbar.

Realtime Streaming

Daten inkrementell nachführen, UI bleibt stabil.

  • Modelle aktualisieren statt neu instanziieren.
  • Repaints gezielt steuern.

Export Pipeline

PNG, SVG oder PDF für Reports und Kunden.

ChartExportService.exportPng(panel, file);
ChartExportService.exportSvg(panel, file);
ChartExportService.exportPdf(panel, file);

Spring Boot

ArberCharts lässt sich problemlos in Spring Boot Services integrieren und als UI-Modul ausliefern.

  • Services liefern Daten, UI rendert live.
  • Ideal für modulare Desktop-Stacks.
